# Requisitos — DNA FERA

## Requisitos Funcionais

### Quiz
- [ ] Formulário multi-etapa (15-20 perguntas)
- [ ] Tipos de pergunta: escolha única, múltipla escolha, numérico
- [ ] Progress bar visual
- [ ] Botão "Voltar" entre etapas
- [ ] Salvamento automático de respostas

### Resultados
- [ ] Geração de perfil de estilo
- [ ] Paleta de cores recomendada
- [ ] Lista de peças sugeridas
- [ ] Links para compra (afiliado)

### Captura de Lead
- [ ] Campo de email obrigatório
- [ ] Opt-in para newsletter
- [ ] Armazenamento em banco de dados

### white-label
- [ ] Configuração de links por criador
- [ ] Subdomínio personalizado (ex: estilo.seunome.com)
- [ ] Logo e cores customizáveis

### Autenticação
- [ ] Login social (Google, Instagram)
- [ ] Perfil do usuário com histórico

## Requisitos Não-Funcionais
- Mobile-first (funcionar bem no celular)
- Velocidade de carregamento < 3s
- SEO otimizado
- SSL/HTTPS

## Integrações
- [ ] API de e-commerce (busca de produtos)
- [ ] Payment gateway (Stripe/Mercado Pago)
- [ ] Email marketing (Mailchimp, RD Station)
- [ ] IA para recomendações (GPT-4, Claude)
- [ ] Análise de imagem (cor da pele)
