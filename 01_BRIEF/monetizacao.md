# Monetização — DNA FERA

## Modelo de Negócio

### 1. Versão white-label Básica (Grátis)
- Quiz completo
- Resultado por email
- Links de produtos do **criador principal** (comissão)
- Sem customização visual
- **Receita:** Comissão de afiliado

### 2. Versão white-label Completa (R$49,90/mês)
- Quiz completo
- Resultado por email
- **Links de produtos do próprio criador** (comissão)
- Customização (logo, cores)
- Subdomínio próprio
- Relatórios avançados
- **Receita:** Assinatura mensal

### 3. Relatório PDF Detalhado (R$19,90)
- Análise completa em PDF
- Paleta de cores personalizada
- Lista de compras recomendada
- **Receita:** Venda avulsa

### 4. Assinatura Premium (R$29,90/mês)
- AI Stylist por chat
- Novos looks mensais
- Programa 12 semanas
- **Receita:** Assinatura mensal

---

## Fluxo de Receita

```
Usuário → Quiz → Email → Resultado → 
    ├── Links afiliado (comissão) ──→ Criador
    ├── Relatório pago ──→ Dono ferramenta
    └── Assinatura ──→ Dono ferramenta
```
